# Demo-Call-Center-
Demo Call Center web app for generating and managing phone numbers with user filters, data saving, Excel export, and an admin dashboard for monitoring and managing users.

Features
 - User login system
 - Network and demographic filters
 - Number generation & data saving
 - Excel export
 - Admin dashboard (add/delete users & track activity)
 - Built as a demo project to simulate real call center operations
